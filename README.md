# Thing Sorter

A web app to sort a custom list of items by user preference, inspired by the Taylor Swift song sorter.

## How to Use

1. Start the server:
   ```
   npm install
   npm start
   ```
2. Open your browser to http://localhost:3000
3. Pick your favorite item in each battle. You can also choose "I Like Both" or "No Opinion".
4. When finished, your sorted list will be displayed.

## Customizing Items

Edit the `items` array in `public/app.js` to use your own list.
